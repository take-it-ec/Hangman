HANGMAN

×

A random word is generated, but it is hidden by blue dashes. the objective of the game is to find the word, letter by letter before the time and before your lives run out

Rules:

  1. Input a letter into the input box provided
  2. if the letter is present in the word,then it is shown on screen.
  3. If the letter isnt present , you will suffer a penalty for every entry, which is each part of man being hung onto the noose.
  4. Find the word by inputting in the correct letters one by one before the poor man is completely hung and before the timer runs out.
  5. Failure in doing so will be Game Over.
  6. But instead you figure out the word then You Win
